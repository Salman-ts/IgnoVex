## ✨ Features

- **Next.js 16 App Router**: Leveraging the latest features for optimal routing, performance, and SEO.
- **React 19**: Modern component architecture with server components support.
- **Tailwind CSS 4**: Highly customizable and utility-first styling.
- **Radix UI Components**: Accessible, unstyled primitives for building high-quality design systems.
- **Framer Motion & tw-animate-css**: Fluid, delightful, and highly optimized micro-animations.
- **Clerk Authentication**: Secure and seamless user authentication and identity management.
- **Zod & React Hook Form**: Type-safe and robust form validation.
- **Responsive & Premium Design**: Mobile-first approach with deep attention to visual excellence.

## 🚀 Getting Started

Follow these steps to run the project locally.

### Prerequisites

Make sure you have Node.js (v20+) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Salman-ts/IgnoVex.git
   cd IgnoVex
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. **Set up Environment Variables:**
   Create a `.env.local` file at the root of the project and provide necessary variables (e.g., Clerk Auth keys).

4. **Run the development server:**
   ```bash
   npm run dev
   ```

5. **Open the app:**
   Navigate to [http://localhost:3000](http://localhost:3000) in your browser.

## 🛠️ Tech Stack

- **Components**: Radix UI
